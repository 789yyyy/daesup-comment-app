# 대나무숲 댓글앱 - 댓글 작성 중 자동 새로고침 방지 버전

수정 내용:
- 댓글을 입력 중이면 1분 자동 새로고침이 잠시 멈춥니다.
- 댓글 입력칸에 작성 중인 내용이 남아 있으면 자동 새로고침으로 내용이 날아가지 않습니다.
- 댓글 등록 후에는 다시 자동 새로고침이 정상 작동합니다.
- 관리자 화면은 PIN 로그인 전에는 관리 도구가 보이지 않습니다.
- 구글폼 닉네임은 게시글에 공개됩니다.

## 배포 방법
1. ZIP 압축을 풉니다.
2. GitHub 저장소에서 `Add file → Upload files`를 누릅니다.
3. 압축 푼 폴더 안의 `netlify`, `public`, `netlify.toml`, `package.json`, `README.md`, `sample_posts.csv`를 전부 올립니다.
4. `Commit changes`를 누릅니다.
5. Netlify 자동 재배포가 끝나면 앱에서 `Ctrl + F5`로 새로고침합니다.
