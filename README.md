## 이번 수정 내용

- 구글시트에 새 줄을 입력하면 앱 새로고침/1분 자동 갱신 때 게시글이 반영됩니다.

## 작동 방식

구글시트에 새 줄을 입력하면 앱이 `/api/feed`를 불러올 때 구글시트 CSV를 읽어서 게시글 목록을 갱신합니다.
앱 화면은 1분마다 자동 새로고침합니다.
댓글은 Netlify Blobs에 저장됩니다.

## 구글시트 권한

Netlify 서버가 시트를 읽을 수 있어야 하므로 구글시트 공유 설정을 아래처럼 해주세요.

1. 구글시트 오른쪽 위 `공유`
2. 일반 액세스
3. `링크가 있는 모든 사용자`
4. 권한은 `뷰어`

## 작성자 닉네임 보안

실제 게시글 작성자 닉네임은 서버에서 일반 이용자에게 내려주지 않습니다.
관리자 PIN을 통과한 요청에서만 실제 닉네임을 내려줍니다.

관리자 화면에서 실제 닉네임을 보려면:

1. 앱 오른쪽 위 `관리자`
2. 관리자 PIN 입력
3. `관리자 정보 보기`
